#### 封装
1. **封装** 是面向对象变成的一大特点
2. 面向对象编程的第一步 -- 将 **属性** 和 **方法** **封装**到一个抽象的类中
3. **外界** 使用 **类** 创建 **对象**，然后让对象调用方法
4. 对象方法的细节 都被**封装**在类的内部

> 一个对象的属性可以是另外一个类创建的对象

#### 需求

2. 士兵可以开火
3. 枪能够发射子弹
