#1.定义变量

###变量定义变量

```
1.定义一个变量记录QQ号码
qq_number='123456'
2.定义一个变量记录QQ密码
qq_password='123'

如果需要通过解释器的方式，输出变量的内容，需要使用print函数
代码：print(qq_number)
```

###用其他变量定义新变量

```
1.定义苹果单价
price=8.5
2.挑选苹果
weight=7.5
3.计算付款金额
money = weight * price

print(money)
结果输出：63.75

#拓展：只要买了5块钱，就返5元
money=money-5 #这里是使用变量，不是定义新变量
print(money)
结果输出：58.75
```
