# 1.定义变量

### 定义变量

```
1.定义一个变量记录QQ号码
qq_number='123456'
2.定义一个变量记录QQ密码
qq_password='123'

如果需要通过解释器的方式，输出变量的内容，需要使用print函数
代码：print(qq_number)
```

### 用其他变量定义新变量

```
1.定义苹果单价
price=8.5
2.挑选苹果
weight=7.5
3.计算付款金额
money = weight * price

print(money)
结果输出：63.75

#拓展：只要买了5块钱，就返5元
money=money-5 #这里是使用变量，不是定义新变量
print(money)
结果输出：58.75
```

# 2.变量的类型

> python定义变量时不需要定义类型

3种数字型数据类型
* ’bool‘--布尔（真假）
* ‘int'--整数
* ’floate‘--浮点数（小数）
* ’复数型‘--用于科学计算，一般用不到

3种型数据类型
* ‘str’--字符串
* 列表
* 元组
* 字典

# 3.不同变量之间的计算

1) 数字型变量之间可以直接计算
2) 字符串变量之间可以用+拼接字符串
3) 字符串变量*整数可以实现批量复制
4) 字符型变量和数字型变量不能进行计算
