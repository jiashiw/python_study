## threading语法

1. 准备一个函数
```
def my_func(a, b):
do_ craw(a, b)
```
2. 怎样创建一个线程
```
import threading
t = threading.Thread(target=my_ func, args=(100, 200))
```
3. 启动线程
```
t.start()
```
4. 等待结束
```
t.join()
```
